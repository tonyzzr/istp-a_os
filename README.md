# MBTI 行为操作系统

为不同人格类型量身定制的生产力框架 | Behavioral Operating Systems for MBTI Types

## 📚 项目简介

本项目提供基于 MBTI 认知功能理论的个性化生产力方法论，包括：

- **ISTP-A 行为操作系统 v3**：专为 ISTP-A 设计的执行系统
- **INFP 行为操作系统**：Fi-Ne-Si-Te-Ti 的柔性执行蓝图

- **框架**：Baseline → Structure → Modules → Pipeline → Entry Point → Snapshot
- **原则**：行动优先 · 边界明确 · 结构轻量
- **案例**：论文写作 · 代码框架 · 求职 · 健身 · PhD Thesis · EB1A/O1

### INFP 系统

- **案例**：面试准备 · 实验计划 · Coding 学习

## 🛠️ 本地开发

1. 克隆项目：
```bash
git clone <repository-url>
cd istp-a_os
```

2. 使用任意 HTTP 服务器运行：
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve

# 使用 PHP
php -S localhost:8000
```

3. 访问 `http://localhost:8000`

## 📦 部署到 Vercel

### 方法一：通过 Vercel CLI

1. 安装 Vercel CLI：
```bash
npm i -g vercel
```

2. 登录并部署：
```bash
vercel login
vercel
```

### 方法二：通过 GitHub 集成

1. 将项目推送到 GitHub
2. 在 [Vercel Dashboard](https://vercel.com/dashboard) 导入项目
3. Vercel 会自动检测并部署
